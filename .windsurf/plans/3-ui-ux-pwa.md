# 🎨 План Покращення UI/UX та PWA

Цей план фокусується на покращенні користувацького досвіду, візуальної складової та перетворенні додатку на повноцінний мобільний продукт.

## 1. Progressive Web App (PWA)
**Мета**: Зробити так, щоб Peak Performer відчувався як нативний додаток на iOS та Android.
- **Кроки**:
  - Інтегрувати `vite-plugin-pwa`.
  - Додати `manifest.json` з іконками різних розмірів, кольором теми та налаштуванням `display: standalone`.
  - Налаштувати Service Workers для кешування ассетів (шрифти, іконки, CSS), щоб додаток миттєво відкривався в офлайні.
  - Додати кастомний екран "Додати на головний екран" (Add to Home Screen).

## 2. Мікро-взаємодії та Анімації (Framer Motion)
**Мета**: Зробити інтерфейс "живим" та приємним у використанні.
- **Кроки**:
  - Плавні переходи між сторінками (Page Transitions) за допомогою `framer-motion`.
  - Додати ефект confetti або приємний "pop" при виконанні "Жаби дня" або завершенні звички.
  - Анімоване згортання/розгортання карток у розділах Бібліотеки та Навчання.

## 3. Темна Тема (Dark Mode) та Доступність
**Мета**: Зручність використання вночі та доступність для всіх.
- **Кроки**:
  - Провести аудит контрастності тексту в Dark Mode.
  - Додати опцію "True OLED Black" (повністю чорний фон замість темно-сірого) для економії заряду на мобільних пристроях.
  - Налаштувати фокусні стани (`focus-visible`) для повноцінної навігації з клавіатури.

## 4. Швидкі Дії (Command Palette)
**Мета**: Прискорити роботу досвідчених користувачів (Power Users).
- **Кроки**:
  - Впровадити Command Palette (натискання `Cmd+K` або `Ctrl+K`), щоб:
    - Швидко додавати завдання, витрати або звички з будь-якого екрану.
    - Швидко перемикатись між розділами.
    - Виконувати пошук по записах журналу та книгах.
